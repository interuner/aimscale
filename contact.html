Before You Send

If you are reporting a conversion issue, include the source game, target game, DPI, sensitivity, result shown, and the result you expected. AimScale can review calculation constants and update pages when better public references are available.

For privacy requests, page corrections, and general feedback, email is the preferred route. GitHub issues are useful when the report includes a reproducible calculator problem.
