Contact

Send a correction or request.

Use this page for calculation feedback, missing game requests, privacy questions, or reports about a page that needs to be updated. The most useful messages include enough detail to reproduce the issue.

Ways To Contact

General feedback contact@aimscale.top
Website https://aimscale.top

What To Include

If you are reporting a conversion issue, include the details below. This makes it possible to compare the calculator result against the formula and the current game constants.

  • Source game and target game.
  • Source sensitivity, source DPI, and target DPI.
  • The result shown by AimScale and the result you expected.
  • The page URL where the issue happened.
  • Any public reference that supports a different yaw value or method.

Request Types

  • Calculator correction: formula, rounding, page copy, or broken input behavior.
  • Missing game request: a game that should be added once reliable public references are available.
  • Privacy request: analytics, contact email, or data handling questions.
  • Page feedback: unclear wording, stale examples, broken links, or mobile layout issues.
  • Business inquiry: advertising, sponsorship, or other site operation questions.

Response Expectations

Email is the preferred route for privacy requests, page corrections, missing game requests, general feedback, and reproducible calculator problems. Do not include passwords, account secrets, payment details, or other private information in feedback messages.

AimScale is a small independent project, so response time can vary. Clear correction reports and privacy requests are prioritized over general feature suggestions.

When a report leads to a page or calculator update, the relevant page should receive a new review date and the change should be reflected in the site checks before deployment.

Common Questions

Can I request a new game?

Yes. Include the game name and any reliable public sensitivity references you found. Games are easier to add when yaw values or conversion behavior can be reviewed.

Can I request a correction?

Yes. Include the page URL, input values, current output, and expected output so the issue can be reproduced.

Where should privacy requests go?

Send privacy requests to contact@aimscale.top and mention that the message is privacy-related in the subject or first sentence.
