# AimScale Agent Rules

AimScale is an FPS mouse sensitivity calculator website.

## Core Rules

- Never change formulas without adding or updating tests.
- Never invent game constants.
- Every game constant should include a confidence level, source references, and a last reviewed date before it is used in production pages.
- Treat conversion results as practical hip-fire cm/360 baselines, not guaranteed identical game feel.
- Every converter page must include a limitation notice and link to `/methodology`.
- Every public page should include a title, meta description, canonical URL, H1, internal links, and a last reviewed or last updated date where appropriate.
- Every new page must be added to `sitemap.xml`.
- Do not place ads above or inside the core calculator.
- Do not publish pro player settings without a source and review date.
- Do not create low-value pages by only replacing game names.

## High-Risk Claims

Flag or rewrite claims using terms such as:

- perfect
- guaranteed
- 100% accurate
- best sensitivity
- proven best

Use phrases like "practical starting point", "hip-fire baseline", or "starting range" when the page is discussing sensitivity recommendations.
